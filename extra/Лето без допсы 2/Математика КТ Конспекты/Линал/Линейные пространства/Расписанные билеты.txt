1. Аксиомы линейного пространства.
    Линейное пространство - X(K) - X-модуль над полем K.
    Аксиомы абелевой группы X:
        1 forall x, y in X: x + y = y + x
        2 forall x, y, z in X: (x + y) + z = x + (y + z)
        3 exist 0: forall x in X: x + 0 = x
        4 forall x in X: exist -x: x + (-x) = 0_X
    Аксиомы поля K.
        1 forall a, b in K: a + b = b + a
        2 forall a, b, c in K: (a + b) + c = a + (b + c)
        3 exist 0: forall a in K: a + 0 = a
        4 forall a in K: exist -a: a + (-a) = 0
        5 forall a, b in K: a * b = b * a
        6 forall a, b, c in K: (a * b) * c = a * (b * c)
        7 exist 1: forall a in K: a * 1 = a
        8 forall a in K\{0}: exist a^(-1): a * a^(-1) = 1
        9 forall a, b, c in K: (a + b) * c = a * c + b * c
    Аксиомы линейного пространства:
        1 forall a, b in K, x in X: (a + b)(x) = a(x) + b(x)
        2 forall a in K, x, y in X: a(x + y) = a(x) + a(y)
        3 forall a, b in K, x in X: a(b(x)) = (a * b)(x)
        4 forall x in X: 1 * x = x
2. Понятие ЛЗ и ЛНЗ набора векторов.
    Набор векторов x1, .., xn называется линейно зависимым, если найдется нетривиальный набор
    коэффициентов a1, .., an, такой что a1 * x1 + .. + an * xn = 0_X.

    Набор x1, .., xn называется линейно независимым, если линейная комбинация этого набора
    равна нулю только при тривиальном наборе коэффициентов.
3. Понятие полного набора. Полнота и линейная независимость.
    Набор называется полным, если любой вектор из X можно представить в виде ЛК этого набора.

    Критерий ЛЗ
        Набор x1, .., xn ЛЗ <=> exist k: xk = +(i = 1..n, i != k, ai * xi)
        =>
            Раз x1, .., xn ЛЗ, то exist k: ak != 0, +(i = 1..n, ai * xi) = 0 =>
            => xk = -+(i = 1..n, i != k, (ai / ak) * xi)
        <=
            exist k: xk = +(i = 1..n, i != k, ai * xi) =>
            => при ak = -1: +(i = 1..n, ai * xi) = 0

    Если набор x1, .., xn полный, то набор x0, x1, .., xn - ЛЗ
        Раз набор x1, .., xn полный, то exist a1, .., an: x = a1 * x1 + .. + an * xn
        Тогда по критерию линейной зависимости набор x0, x1, .., xn ЛЗ.

    Процедура замещения
        Если y1, .., yn - полный набор, то forall x != 0_X in X(K) exist k: y1, .., y[k - 1],
        y[k + 1], .., yn, x - полный набор.
            Можем выразить x как x = a1 * y1 + .. + an * yn, при этом exist k: ak != 0 =>
            => yk = 1 * x - +(i = 1..n, i != k, (ai / ak) * yi)
            При этом forall z in X(K): z = b1 * y1 + .. + bn * yn = +(i = 1..n, i != k,
            bi * yi) + bk * yk = +(i = 1..n, i != k, bi * yi) + bk * (1 * x - +(i = 1..n,
            i != k, (ai / ak) * yi)). Тогда приведем подобные слагаемые и получим, что
            мы можем выразить любой z in X(K) с помощью {y1, .., yn, x}\{yk}.

    Число векторов ЛНЗ набора не превосходит числа векторов полного набора
        x1, .., xm - ЛНЗ набор, y1, .., yn - полный набор => m <= n
            Пусть m > n
            Воспользуемся процедурой замещения векторов в полном наборе
            Тогда на каждом из n шагов будем добавлять в набор xi, и находить yj, который
            можно представить в виде ЛК остальных векторов, а затем вычеркивать этот yj.
            При этом мы обязательно будем вычеркивать один элемент из набора y, ведь все
            элементы из x ЛНЗ => мы не сможем выразить xi только через x1, .., x[i - 1],
            а значит найдется yj с ненулевым коэффициентом. Тогда после n шагов получили,
            что набор x1, .., xn полный, но при этом мы не можем выразить через него x[n + 1],
            .., x[m], тк по условию набор x1, .., xm ЛНЗ.
4. Базис и размерность линейного пространства.
    Базисом в линейном пространстве X(K) называется полный ЛНЗ набор.

    Размерностью линейного пространства называется мощность его базиса.
5. Координаты вектора, единственность координат.
    Пусть e1, .., en - базис ЛП X(K). Тогда forall x in X(K): exist a1, .., an: x = a1 * xn +
    + an * en, т.к. базис это ПН. При этом a1, .., an называются координатами вектора x в
    базисе e1, .., en.

    Координаты любого вектора из X(K) в заданном базисе определяются единственным образом.
        Пусть не так. Тогда exist x in X(K): a1 * e1 + .. + an * en = x = b1 * e1 + .. + bn *
        * en => e1 * (a1 - b1) + .. + en * (an - bn) = 0, но раз базис ЛНЗ, то ЛК тривиальна
        только при тривиальном наборе коэффициентов => forall i = 1..n: ai = bi.
6. Изоморфизм линейных пространств.
    Изоморфизмом называется отношение f: X(K) -> Y(K), которое биективно и линейно.

    Отображение f: X -> Y называется биективным, если существует отображение g: Y -> X, такое
    что: forall x in X: g(f(x)) = x, forall y in Y: f(g(y)) = y.

    Отображение f: X(K) -> Y(K) называется линейным, если:
        forall x, y in X: f(x + y) = f(x) + f(y)
        forall x in X, a in K: f(a * x) = a * f(x)
7. Изоморфизм - отношение эквивалентности.
    Пусть X(K) - ЛП, e1, .., en - базис X(K). Тогда отображение f: X(K) -> K^n, сопоставляющее
    каждому вектору набор его координат в базисе e1, .., en, является изоморфизмом.
        Понятно, что forall a1, .., an in K^n exist x in X(k): x = a1 * e1 + .. + an * en.
        Также мы доказали, что forall x in X(k) exist! a1, .., an: x = a1 * e1 + .. + an * en.
        Раз наше отображение инъективно и сюръективно, то оно биективно.
        Тогда осталось проверить, что оно линейно.
        1 f(x) + f(y) = (a1, .., an)^T + (b1, .., bn)^T = (a1 + b1, .., an + bn) = f(x + y)
        2 f(a * x) = (a * b1, .., a * bn) = a * (b1, .., bn) = a * f(x)

    Пространства X(K) и Y(K) изоморфны, если существует изоморфизм f: X(K) -> Y(K).
    Докажем, что изоморфность - отношение эквивалентности.
    1 Рефлексивность X изоморфно X
        id(X): X -> X - изоморфизм
    2 Симметричность X изоморфно Y => Y изоморфно X
        Для любого изоморфизма f: X -> Y существует обратный изоморфизм g: Y -> X
    3 Транзитивность X изоморфно Y, Y изоморфно Z => X изоморфно Z.
        Пусть f: X -> Y - изоморфизм, g: Y -> Z - изоморфизм
        Тогда g◦f: X -> Z - изоморфизм

8. Критерий изоморфности двух пространств.
    X(K) изоморфно Y(K) <=> dim(K, X) = dim(K, Y) (размерности X(K) и Y(K) совпадают)
        =>
        Раз X(K) изоморфно Y(K), то какой-то базис X(K) отобразится в какой-то базис Y(K),
        при этом размеры базисов будут равны, ведь forall i ei -> f(ei), где e1, .., en -
        базис X(K), f(e1), .., f(en) - базис Y(K).
        Базис отразится в базис, потому что forall y in Y: y = f(x) = f(a1 * e1 + .. + an *
        * en) = a1 * f(e1) + .. + an * f(en) => можно выразить любой элемент из Y(K) с
        помощью ЛК f(e1), .., f(en) => это ПН. При этом f(e1), .. f(en) ЛНЗ, а иначе
        пусть a1 * f(e1) + .. + an * f(en) = 0 => f(a1 * e1 + .. + an * en) = 0 =>
        => a1 * e1 + .. + an * en = 0 => a1 = .. = an = 0, т.к. набор e1, .., en - ЛНЗ.
        Ноль отображается в ноль, т.к. f(0) = f(x - x) = f(x) - f(x) = 0.
        <=
        Пусть dim(K, X) = n = dim(K, Y). Тогда X(K) изоморфно K^n, Y(K) изоморфно K^n =>
        K^n изоморфно Y(K) по симметричности => X(K) изоморфно Y(K) по транзитивности.
9. Подпространство линейного пространства.
    Подмножество L множества X называется ЛПП ЛП X(K), если оно само является ЛП над полем К
    относительно операций, определенных в X.

    Критерий ЛПП
    Пусть L - непустое подмножество множества X

    L(K) - ЛПП
    <=>
    1 forall x1, x2 in L: x1 + x2 in L
    2 forall a in K, x in L: a * x in L
        =>
        L(K) - ЛПП => эти два пункта есть в аксиомах ЛПП
        <=
        Можно проверить аксиомы. Понятно, что аксиомы поля K выполняются.
        Аксиомы группы выполняются, тк они выполнялись для X, и при этом L замкнуто
        относительно операции, индуцированной из X.
        Аксиомы ЛП
        (a * b) * x = a * (b * x) - выполняется для X(K) => для L(K), при этом обе части
        равенства существуют, исходя из второго пункта
        a * (x + y) = a * x + a * y - выполняется для X(K) => для L(K), обе части существуют
        (a + b) * x = a * x + b * x - выполняется для X(K) => для L(K), обе части существуют
        1 * x = x - выполняется для X(K) => для L(K).
10. Линейная оболочка - линейное подпространство.
    Линейная оболочка векторов x1, .., xn - множество всех ЛК этих векторов.

    Пусть L - ЛО x1, .., xn. Тогда L - линейное подпространство X(K).
    forall a in K, x in L: a * x in L <=> a * (a1 * x1 + .. + an * xn) = (a * a1) * x1 + .. +
    + (a * an) * xn in L.
    forall x, y in L: x + y in L <=> (a1 * x1 + .. + an * xn) + (b1 * x1 + .. + bn * xn) in L
    <=> (a1 + b1) * x1 + .. + (an + bn) * xn in L.
    Оба пункта являются правдой, ведь финальные выражения - это ЛК векторов x1, .., xn.
11. Понятие линейного многообразия.
    Линейным многообразием М, параллельным подпространству L линейного пространства X(K)
    называется множество M = {x0 + x, x0 in X(K), x in L(K)}, x0 фиксирован, берутся все
    x in L(K).
12. Пересечение и сумма линейных подпространств.
    Пусть L1(K), L2(K) - подпространства X(K)
    Тогда L'(K) = {x in X(K): x in L1(K) && x in L2(K)} - пересечение подпространств L1 и L2.
    А L''(K) = {x1 + x2: x1 in L1(K), x2 in L2(K)} - сумма подпространств L1 и L2.
13. Теорема Грассмана: размерности суммы и пересечения подпространств.
    Пусть L1 и L2 - ЛПП X(K).
    Тогда dim(K, L1) + dim(K, L2) = dim(K, L1 + L2) + dim(K, L1 && L2)
    Доказательство
    Если перенести dim(K, L1 && L2) вправо, то можно сформулировать теорему так:
    Пусть {e1, .., em} - базис L1 && L2, {e1, .., em, f1, .., fk} - базис L1, {e1, .., em,
    g1, .., gl} - базис L2. Тогда {e1, .., em, f1, .., fk, g1, .., gl} - базис L1 + L2.
    Тогда нужно показать, что этот набор ПН и ЛНЗ в L1 + L2.
    Набор полный, тк forall x in L1 + L2: exist x1 in L1, x2 in L2: x = x1 + x2 = (a1 * e1 +
    + .. + am * em + a[m + 1] * f1 + .. + a[m + k] * fk) + (b1 * e1 + .. + bm * em +
    + b[m + 1] * g1 + .. + b[m + l] * gl) = (a1 + b1) * e1 + .. + (am + bm) * em + a[m + 1] *
    * f1 + .. + a[m + k] * fk + b[m + 1] * g1 + .. + b[m + l] * gl, то есть мы можем
    представить любой x in L1 + L2 с помощью ЛК {e1, .., em, f1, .., fk, g1, .., gl} => это ПН
    Теперь надо доказать, что набор {e1, .., em, f1, .., fk, g1, .., gl} - ЛНЗ.
    Рассмотрим произвольную тривиальную ЛК этого набора:
    a1 * e1 + .. + am * em + b1 * f1 + .. + bk * fk + c1 * g1 + .. + cl * gl = 0 <=>
    a1 * e1 + .. + am * em + b1 * f1 + .. + bk * fk = z = -c1 * g1 + .. + -cl * gl
    Тогда z лежит в L1, т.к. может быть представлен в базисе L1, а также лежит в L2, ведь
    может быть представлен в базисе L2. Тогда z in L1 && L2 => z = d1 * e1 + .. + dm * em
    Тогда d1 * e1 + .. + dm * em + c1 * g1 + .. + cl * gl = 0 => d1 = 0, .., dm = 0, c1 = 0,
    .., cl = 0, т.к. {e1, .., em, g1, .., gl} - ЛНЗ. Тогда z = 0 => a1 * e1 + .. + am * em +
    + b1 * f1 + .. + bk * fk = 0 => a1 = 0, .., am = 0, b1 = 0, .., bk = 0, ведь {e1, .., em,
    f1, .., fk} - ЛНЗ. Тогда получили, что тривиальная комбинация может быть только при
    тривиальном наборе коэффициентов => {e1, .., em, f1, .., fk, g1, .., gl} - ЛНЗ.
    Раз этот набор ПН и ЛНЗ, то это базис пространства L1 + L2.
14. Прямая сумма подпространств. Критерий прямой суммы.
    Прямой суммой подпространств L1 и L2 пространства X называется множество {x1 + x2, x1 in
    L1, x2 in L2}, если для каждого элемента x in L = L1 + L2 существует единственное
    разложение на сумму векторов.
    Критерий прямой суммы
    L1 + L2 = L1 +. L2 <=> L1 && L2 = {0_X}
    Доказательство
    =>
    От противного, пусть L1 + L2 - прямая сумма, но L1 && L2 != {0_X}
    Пусть z in L1 && L2, z != 0_X => forall x in L1 + L2: x = x1 + x2 = x1 + x2 + z - z =
    = (x1 + z) in L1 + (x2 - z) in L2 => можно представить x не единственным образом, что
    противоречит определению прямой суммы.
    <=
    От противного, пусть L1 && L2 = {0_X}, exist x in L1 + L2: x1 + x2 = x = y1 + y2 =>
    => x1 - y1 = z = y2 - x2, z in L1 && L2 => z = 0_X => x1 = y1, x2 = y2
15. Разложение линейного пространства в прямую сумму подпространств.
    ЛПП L = +.(i = 1..k, Li) называется прямой суммой подпространств L1, .., Lk, если
    forall x in L: exist! {x1, .., xk}: x = x1 + .. + xk.

    Лемма. Сумма ЛПП является прямой => forall i=1..k, j != i: Li && Lj = {0_X}.
    От противного, пусть exist i, j != i: Li && Lj != {0_X}
    Пусть z in Li && Lj, z != 0. Тогда forall x in L: x = x1 + .. + xi + .. + xj + .. + xk =
    = x1 + .. + xi + .. + xj + .. + xk + z - z = x1 + .. + (xi + z) + .. + (xj - z) + .. + xk.
    То есть мы нашли два разложения для одного x in L, что противоречит определению прямой
    суммы ЛПП.

    Критерий разложения ЛП X(K) в прямую сумму ЛПП L1, .., Lk.
    X = +.(i = 1..k, Li) <=> forall i = 1..k, j != i: Li && Lj = {0_X}, dim(K, X) = +(i =
    = 1..k, dim(K, Li)).
    Доказательство
    =>
    dim X = dim +.(i = 1..k, Li) => dim X =  dim (L_k +. Lk) = dim (L_k) + dim(Lk), т.к. L_k -
    ЛПП X, Lk - тоже ЛПП X, при этом их сумма равна прямой сумме => dim L_k + Lk = dim L_k +
    + dim Lk - (dim(L_k && Lk) = 0). То есть их пересечение тривиально, т.к. их сумма равна
    прямой сумме. Тогда мы можем просто раскрыть скобки и получить dim X = dim L1 + .. + dim
    Lk.
    <=
    Имеем: dim X = dim L1 + .. + dim Lk
    Так же знаем, что dim L1 + .. + Lk = dim L1 + .. + L[k - 1] + dim Lk - dim (L1 + .. + L[k
    - 1]) && Lk = dim L1 + .. + L[k - 1] + dim L[k - 1] + dim Lk - dim (L1 + .. + L[k - 1]) &&
    && Lk - dim (L1 + .. + L[k - 2]) && L[k - 1] = .. = dim L1 + .. + dim Lk - dim (L1 && L2)
    - dim (L1 + L2) && L3 - .. - dim (L1 + .. + L[k - 1]) && Lk <= dim X
16. Проекция на подпространство. Понятие проектора.
    Пусть X = L1 +. L2. Тогда forall x in X: x = x1 in L1 + x2 in L2.
    При этом x1, x2 - компоненты x в L1 и L2
    x1 = P(||L2, L1, x) - проeкция x на L1 параллельно L2
    x2 = P(||L1, L2, x) - проекция x на L2 параллельно L1
    P(||L2, L1, x) - проектор на пространство L1
    P(||L1, L2, x) - проектор на пространство L2
    L1 - дополнение L2 до X
    L2 - дополнение L1 до X
17. Дополнение линейного подпространства. Коразмерность.
    Если X = L1 +. L2, то L1 - дополнение L2 до X, L2 - дополнение L1 до X.
    Размерность дополнения ЛПП до ЛП называется его коразмерностью
        dim(K, X) = n, dim(K, L) = k => codim(K, L) = n - k
18. Линейная форма как отображение.
    Линейной формой f на ЛП X(K) называется отображение f: X(K) -> K, обладающее свойством
    линейности, то есть для которого верно:
        f(x + y) = f(x) + f(y) forall x, y in X(K)
        f(a * x) = a * f(x) forall x in X(K)
19. Пространство линейных форм.
    Говорят, что линейные формы f и g равны(f = g), если
        forall x in X(K): (f, x) = (g, x)
    Линейная форма называется нулевой, если
        forall x in X(K): (O, x) = 0
    Суммой линейных форм f и g называется отображение h = f + g:
        forall x in X(K): (h, x) = (f, x) + (g, x)
    Произведением линейной формы f на число a in K называется отображение g = a * f:
        forall x in X(K): (g, x) = a * (f, x)
    Тогда множество X*(K) линейных форм на X(K) является ЛП.
    Проверим аксиомы:
    Аксиомы абелевой группы X*:
        1 forall f, g in X*: x + y = y + x
        2 forall f, g, h in X*: (x + y) + z = x + (y + z)
        3 exist O: forall f in X*: f + O = f
        4 forall f in X*: exist -f: f + (-f) = O
    Аксиомы поля K соблюдены, т.к. оно не менялось.
    Аксиомы линейного пространства:
        1 forall a, b in K, f in X*: (a + b)(f) = a(f) + b(f)
        2 forall a in K, f, g in X*: a(f + g) = a(f) + a(g)
        3 forall a, b in K, f in X*: a(b(f)) = (a * b)(f)
        4 forall f in X*: 1 * f = f
20. Теорема о базисе линейного пространства форм.
    Коэффициентами линейной формы в базисе {e1, .., en} ЛП X(K) называются ее значения на
    базисных векторах, т.е. f(ei) = ai, f <-> {a1, .., an}.

    Теорема о базисе X*.
    Множество линейных форм f1, .., fn : X(K) -> K, действующее с базисом X(K) e1, .., en,
    как fi(x) = ai, x = a1 * e1 + .. + an * en, образует базис пространства X*.
    Доказательство
    1 Полнота
    (f, x) = (f, a1 * e1 + .. + an * en) = a1 * (f, e1) + .. + an * (f, en) = (f1, x) *
    * (f, e1) + .. + (fn, x) * (f, en) = (f1, x) * b1 + .. + (fn, x) * bn - ЛК {f1, .., fn}
    2 ЛНЗ
    a1 * f1 + .. + an * fn = O => forall x in X(K) (a1 * f1 + .. + an * fn, x) = 0 =>
    => a1 * (f1, x) + .. + an * (fn, x) = 0 => a1 = a2 = .. = an = 0, иначе пусть ak != 0,
    a1 * (f1, x) + .. + an * (fn, x) = 0, тогда a1 * (f1, x + ek) + .. + an * (fn, x + ek) !=
    != 0.
21. Сопряженное пространство. Сопряженные базисы.
    Сопряженным пространством пространства X(K) называется пространство X*(K) линейных
    форм на X(K).

    Базисы e1, .., en, f1, .., fn пространств X(K) и X*(K) называются сопряженными, если
    (fi, ej) = 1 <=> i = j.
22. Второе сопряженное пространство. Естественный изоморфизм.
    Можно ввести пространство X**(K) - пространство линейных форм на X*(K).
        x_: X*(K) -> K
        (x_, f + g) = (x_, f) + (x_, g) forall x_ in X**(K), f, g in X*(K)
        (x_, a * f) = a * (x_, f) forall x_ in X**(K), f in X*(K)

    Изоморфизм двух линейных пространств называется естественным изоморфизмом, если он
    устанавливается без применения понятия базиса.

    Лемма
    Изоморфизм между пространствами X**(K) и X(K) - естественный.
    Доказательство
    Искомый изоморфизм x_ <-> x, (x_, f) = (f, x) forall f in X*(K)
    Тогда на X**(K) естественным образом индуцируется структура ЛП

23. Классификация систем линейных алгебраических уравнений.
    Система вида
    a[1][1] * x1 + .. + a[1][n] * xn = b1
    ...
    a[m][1] * x1 + .. + a[m][n] * xn = bm
    Называется линейной алгебраической системой из m уравнений и n неизвестных. При этом
    a[i][j] - коэффициенты системы, xi - неизвестные, bi - свободные члены.

    Решением системы называется набор x1, .., xn, при подстановке которого получаются верные
    равенства.

    Система называется совместной, если она имеет решение, иначе называется несовместной.

    Система называется определенной, если она совместна и имеет единственное решение, иначе
    она называется неопределенной.

    Если все свободные члены равны нулю, то система называется однородной, иначе неоднородной.

    Альтернативные формы записи:
    1 Матричная A * X = B,
        A = {
            a[1][1], .., a[1][n];
            a[2][1], .., a[2][n];
            ..;
            a[m][1], .., a[m][n]
        }
        X = {x1, .., xn}^T
        B = {b1, .., bm}^T
    2 Векторная a1 * x1 + .. + an * xn = b,
        ai = {a[i][1], .., a[i][m]}^T
        b = {b1, .., bm}^T
        Тогда нахождение решения <=> нахождению коэффициентов x1, .., xn линейной комбинации
        a1, .., an.
24. Теорема Кронеккера-Капелли. Системы Крамера.
    Линейная алгебраическая система называется системой Крамера, если m = n и набор векторов
    a1, .., an - ЛНЗ.

    Пусть система задана в виде a1 * x1 + .. + an * xn = b, причем dim(K, <a1, .., an>) = r
    <= m, тогда перенумеруем переменные так, чтобы первые r векторов a1, .., ar были ЛНЗ,
    то есть базисными для линейной оболочки <a1, .., an>, а остальные выражались через них.
    В новой нумерации можем записать a1 * x1 + .. + ar * xr = b - a[r + 1] * x[r + 1] - .. -
    - an * xn.

    Теорема Кронеккера-Капелли
    Система a1 * x1 + .. + ar * xr = b - a[r + 1] * x[r + 1] - .. - a[n] * x[n] совместна
    тогда и только тогда, когда b принадлежит линейной оболочке <a1, .., ar>. При этом если
    r = n, то система определена, иначе неопределена.
    Доказательство
    =>
    Система совместна => мы можем выразить все вектора a[r + 1], .., a[n] с помощью базисных
    и перенести налево полученные выражения. Тогда получим a1 * x1 + .. + ar * xr = b =>
    => b in <a1, .., ar>, тк является ЛК этих векторов.
    <=
    Раз b in <a1, .., ar>, то мы можем представить b как ЛК базисных векторов, при этом
    x[r + 1], .., x[n] можно сделать равными нулю и тогда exist x1, .., xr: a1 * x1 + .. +
    + ar * xr = b.
    При этом если r = n, то мы сможем представить b в виде ЛК векторов x1, .., xr единственным
    образом, т.к. координаты вектора в базисе единственны.
    Если же r < n, то любую ЛК векторов a[r + 1], .., a[n] мы тоже сможем представить в виде
    ЛК векторов a1, .., ar, а значит для любой из них сможем найти решение системы, то есть
    система будет неопределена.
25. Теорема Кронеккера-Капелли: однородные системы.
    Следствия:
    Однородная система всегда совместна, т.к. существует тривиальное решение, где xi = 0.
    Однородная система имеет нетривиальные решения тогда и только тогда, когда r < n.
    Является неопределенной тогда и только тогда, когда r < n.
26. Структура общего решения однородной СЛАУ.
    Пусть S - множество решений однородной СЛАУ. Тогда S - ЛПП K^n.
    Доказательство
    Достаточно показать, что forall x1, x2 in S: x1 + x2 in S, forall a in K, x in S: a * x in
    S.
    Пусть x1, x2 in S. Тогда x1 = a1 * e1 + .. + an * en, x2 = b1 * e1 + .. + bn * en
    Раз x1 - решение однородной СЛАУ, то c1 * a1 + .. + cn * an = 0, то же верно и для x2,
    то есть c1 * b1 + .. + cn * bn = 0. Тогда c1 * (a1 + b1) + .. + cn * (an + bn) = 0 =>
    x3 = (a1 + b1, .., an + bn) = x1 + x2 тоже является решением СЛАУ => x1 + x2 in S.
    Раз x in S, то x = a1 * e1 + .. + an * en, при этом x - решение СЛАУ => c1 * a1 + .. +
    + cn * an = 0. Домножив левую и правую часть на a получим c1 * (a * a1) + .. + cn *
    * (a * an) = 0 => a * x тоже решение СЛАУ => a * x in S.
27. Фундаментальная система решений.
    Фундаментальной системой решений СЛАУ называется любая система из n - r ЛНЗ решений этой
    СЛАУ, то есть базис пространства решений этой системы.
28. Неоднородная система линейных уравнений: структура общего решения.
    Множество решений неоднородной СЛАУ является линейным многообразием, ведь мы можем найти
    частное решение этой системы, а затем прибавить к нему любую ЛК не базисных
    векторов-коэффициентов перед неизвестными, то есть решение неоднородной СЛАУ есть сумма
    частного решения и ЛК векторов из ФСР однородной системы, записанной по данной.
