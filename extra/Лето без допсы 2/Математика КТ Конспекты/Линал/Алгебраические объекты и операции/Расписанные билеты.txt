1. Закон композиции: определение, примеры

    Внутренним законом композиции на множестве M называется отображение
    M × M → M декартова произведения M × M в M. Значение (x, y) |-> z in M
    называется композицией элементов x и y относительно этого закона.

    Примеры: сложение R^2 -> R, умножение R^2 -> R

2. Закон композиции: ассоциативность, коммутативность

    Закон композиции элементов множества M называется ассоциативным, если для любых элементов x, y, z in M выполняется равенство:
    (x ◦ y) ◦ z = x ◦ (y ◦ z).

    (теорема об ассоц законе

    Элементы x, y in M называются перестановочными относительно заданного закона композиции, если имеет место равенство: x ◦ y = y ◦ x.
    Если перестановочна любая пара элементов x, y in M, тогда внутренний закон ◦ называется коммутативным.

    Необязательные примеры: на Z[1/2] (x + y)/2 - неассоциативный, а НОД(x, y) - ассоц.

3. Основные структуры с одним законом композиции

    Магма - множество, наделенное внутренним законом композиции.

    Полугруппа - множество M, наделенное ассоциативным всюду определенным законом композиции.

    Моноид - полугруппа S, содержащая нейтральный элемент.

    Группа - множество, наделенное внутренним законом композиции,
    удовлетворяющим следующим требованиям (аксиомам):
        1. ассоциативность закона
        2. существование нейтрального элемента 
        3. для каждого элемента существует обратныйФ

4. Группа: определение, примеры

    Непустое множество G называется группой, если на нем задан закон композиции
    G × G → G, так что (x, y) |→ xy и имеют место следующие три свойства:
    G1. Ассоциативность закона:
        ∀x, y, z ∈ G (xy)z = x(yz)
    G2. Существует нейтральный элемент:
        ∃ e ∈ G : ∀x ∈ G xe = x = ex
    G3. Существует обратный элемент:
        ∀x ∈ G ∃ x^−1: xx^−1 = e = x^−1x

    Примеры групп: множество вещественных чисел без 0 по умножению, множество целых по сложению, {0} по сложению

5. Гомоморфизмы групп: определение, свойства

    Гомоморфизмом групп G и G′ называется отображение σ : G → G′, обладающее
    следующими свойствами: ∀x, y ∈ G σ(xy) = σ(x)σ(y)

    Свойства:
        1. Пусть σ in Hom(G, G′), тогда σ(e) = e'.
            σ(e) = σ(e*e) = σ(e)*'σ(e) <=> σ(e) = σ(e)*'σ(e) <=> e' = σ(e)
        2. Пусть σ in Hom(G, G′), тогда ∀x ∈ G σ(x^−1) = σ(x)^−1.
            σ(x^-1) = σ(x)^-1 <=> σ(x^-1)σ(x) = σ(x)^-1σ(x) <=> σ(x^-1x) = e' <=> σ(e) = e' <=> 1
        3. Ядро ker σ является группой.
            ker σ = {x: σ(x) = e'}
            Возьмем закон из G
            1) Он ассоциативен по аксиомам группы
            2) σ(e) = e' => e in ker σ
            3) Пусть x in ker σ: e' = σ(e) = σ(x^-1x) = σ(x^-1)σ(x) = σ(x^-1)e' = σ(x^-1) => x^-1 in ker σ
        4. Гомоморфизм σ in Hom(G, G′), ядро которого тривиально, иньективен.
            ker σ = {e} => forall σ(x) = σ(y) => x = y
            σ(x) = σ(y) <=> σ(xy^-1) = e' <=> xy^-1 = e <=> x = y
        5. Образ Im σ является группой
            Im σ = {σ(x) : x in G}
            Возьмем закон из G'
            1) Он ассоциативен по аксиомам группы
            2) σ(e) = e' => e' in Im σ
            3) forall x in Im σ exist x^-1
                x = σ(y)
                σ(y^-1) = σ(y)^-1 = x^-1

6. Подгруппы: определение, примеры

    Подгруппой H группы G называется подмножество G, имеющее структуру группы, индуцированной групповым законом G.

    Пусть σ in hom(G, G′), тогда ker σ ≤ G и Im σ ≤ G′.

7. Нормальная подгруппа, фактор-группа

    Подгруппа H группы G называется нормальной, если forall x in G xH = Hx = x_.

    Группа G/H называется фактор-группой группы G по нормальной подгруппе H. Она является группой всех классов
    эквивалентности, получающихся применением отношения эквивалентности ~: x~y <=> xy^-1 in H <=> x in Hy.

8. Понятие канонического гомоморфизма

    Теорема: пусть H норм подгр G, тогда существует такой гомоморфизм φ (называемый каноническим), что ker φ = H.
        Рассмотрим φ: G -> G/H, φ(x) = xH
        1) φ(x)φ(y) = xHyH = xHHy = xHy = xyH = φ(xy)
        2) φ(e) = eH = H
        Т.е. φ - гомоморфизм
        x in ker φ <=> φ(x) = H <=> xH = H <=> x~e <=> xe^-1 in H <=> x in He <=> x in H

9. Теорема об изоморфизме для групп

    Теорема об изоморфизме: Пусть σ : G → G′ - гомоморфизм групп, тогда Iso(G/ker σ,  Im σ) != {}.
        Зададим отображение σ_ : G/ker σ → Im σ, σ_(x_) = σ(x), и покажем, что оно определено корректно.
        Именно, пусть x_ = y_(=> xy^-1 in ker σ, (xy^-1)^-1 in ker σ),
        тогда σ_(y_) = σ(y) = σ(xx^−1y) = σ(x)σ(x^−1y) = σ(x)e = σ(x) = σ_(x_).
        Далее, σ_ - гомоморфизм: σ_((xy)_) = σ(xy) = σ(x)σ(y) = σ_(x_)σ_(y_).
        Тривиально проверяется, что Im σ_ = Im σ:
            x' in Im σ <=> exist x in G: σ(x) = σ_(x_) = x'
        , и остается прямой проверкой убедиться, что ядро σ_ тривиально(по св-вам гомоморфизма):
        z_ in ker σ_ => σ(z) = σ_(z_) = e => z in ker σ => z_ = e_.
    Таким образом, мы показали, что σ_ - изоморфизм.

10. Два закона композиции на множестве

    Пусть + и * - законы композиции на множестве M.
    Закон * дистрибутивен слева по закону +, если forall x, y, z in M x*(y + z) = x*y + x*z
    Закон * дистрибутивен справа по закону +, если forall x, y, z in M (x + y)*z = x*z + y*z
    Если * дистрибутивен по + и слева, и справа, то он двояко-дистрибутивен по +.

    Пусть на M заданы законы + и *, причем * наделяет М структурой группы
    и есть нейтральный элемент e по +, а также * дистрибутивно по +.
    Тогда x*e = e: x*e = x*(e + e) = x*e + x*e <=> x*e + -(x*e) = e = x*e = x*e + x*e + -(x*e).
    Еще можно записать, что x*y = e + x*y, x*y = x*(e + y) = x*e + x*y <=> x*e + x*y = x*y = e + x*y.

11. Структура коммутативного кольца: определение, примеры

    Аксиомы кольца:
        1) Коммутативное и 2) ассоциативное сложение 3) с противоположным и
        4) нейтральным элементом, а также 5) ассоциативное и
        6) двояко-дистрибутивное по сложению умножение.
    Коммутативное кольцо - кольцо с коммутативностью умножения.
    Кольцо с единицей - кольцо с нейтральным элементом по умножению.

    Примеры:
        Нулевое кольцо
        Целые числа
        Z[1/2] = {m/2^n: m in Z, m не кратно 2, n in N}
        Z[sqrt(2)] = {x + sqrt(2)*y: x, y in Z}
        Z[i] = {x + i*y, x, y in Z, i^2 = -1}
        Z[x] = {+(0, +inf)(a[i]*x^i), a[i] in Z}, Z[x1, .., xn]
        Кольцо матриц - некоммутативное кольцо.

12. Гомоморфизмы колец: определения, свойства

    Пусть R, R' - кольца. Тогда f: R -> R' - гомоморфизм, если:
        1) f(x + y) = f(x) +' f(y)
        2) f(x*y) = f(x)*'f(y)
        3) f(1) = 1'

    Пусть A, B, C - кольца, f: A -> B, g: B -> C - кольцевые гомоморфизмы,
    тогда g◦f: A -> C тоже кольцевой гомоморфизм:
        1) (g◦f)(x + y) = g(f(x + y)) = g(f(x) +' f(y)) = g(f(x)) +'' g(f(y))
        2) (g◦f)(x * y) = g(f(x * y)) = g(f(x) *' f(y)) = g(f(x)) *'' g(f(y))
        3) (g◦f)(1) = g(f(1)) = g(1') = 1''

13. Подкольцо: определение, примеры

    Подкольцом кольца R называется абелева подгруппа R по +, содержащая 1.

14. Идеал кольца, фактор-кольцо

    Идеалом J кольца R называется аддитивная подгруппа, для которой верно:
        RJ ( J <=> forall x in R, y in J: x*y in J

    Любой идеал кольца целых чисел имеет вид nZ, n in N.
    Предположим, что не так. Найдем наименьшее целое число n,
    принадлежащее идеалу I. Если каждое число в I делится на n,
    то перед нами I = nZ, иначе найдем число a, не делящееся на n
    и рассмотрим его: a = n*(k in Z) + r, 0 < r < n. (n in I)*(k in Z) in I,
    тогда r = (a in I - n*k in I) in I, т.к. I - аддитивная подгруппа,
    поэтому для любых двух элементов из I результат их сложения тоже в I.
    Но это противоречие с выбором n.

    Фактор-кольцом R/J кольца R по идеалу J называется множество классов
    эквивалентности, полученных отношением эквивалености ~: x~y <=> x - y in J.
    Каждый такой класс имеет вид x_ = x + J.

    Эта структура с операциями + и * из R будет кольцом, если:
        1) x_ + y_ = (x + y)_
            x_ + y_ = x + J + y + J = x + y + J = (x + y)_
        2) x_*y_ = (x*y)_
            x_*y_ = (x + J)*(y + J) = x*y + (x*J in J) + (y*J in J) + (J*J in J) = (x*y)_
        3) 0_ = J
            0_ + x_ = J + x + J = x + J = x_
            0_*x_ = J*(x + J) = J*x in J + J*J in J = J = 0_


15. Теорема об изоморфизме колец

    Пусть f: A -> B - гомоморфизм колец A и B. Тогда ker f - идеал A,
    Im f - подкольцо B, A/ker f изоморфно Im f.
        Покажем, что ker f - идеал A.
            x in ker f <=> f(x) = 0 => forall y in A: f(x*y) = f(x)*'f(y) = 0 =>
            forall y in A x*y in ker f <=> A*ker f ( ker f
        Im f - подкольцо B, т.к. Im f содержит 1 и является абелевой подгруппой B по +.
        Рассмотрим отображение g: A/ker f -> Im f, g(x_) -> f(x).
            1) g(x_ + y_) = g((x + y)_) = f(x + y) = f(x) + f(y) = g(x_) + g(y_)
            2) g(x_*y_) = g((x*y)_) = f(x*y) = f(x)*f(y) = g(x_)*g(y_)
            3) g(1_) = f(1) = 1
            То есть g - гомоморфизм.
            forall x in Im f exist f(y) = x => exist g(y_) = f(y)
            То есть g - сюръекция.
            g(x_) = g(y_) <=> f(x) = f(y) <=> f(x - y) = 0 = g(x_ - y_) <=> x - y in ker f <=> x~y <=> x_ = y_

16. Нильпотенты и обратимые элементы кольца

    Нильпотентом называется элемент x != 0 кольца R: exist n in N: x^n = 0.

    Элемент x кольца R называется обратимым, если exist y: x*y = 1.

17. Область целостности. Поле

    Областью целостности называется кольцо, в котором нет делителей нуля, т.е.
        forall x in R\{0} forall y in R\{0} x*y != 0

    Полем называется ненулевое кольцо, каждый ненулевой элемент которого обратим.
    Т.е. аксиомы поля: сложение коммутативно, ассоциативно, есть противоположный и
    нейтральный элементы; умножение коммутативно, ассоциативно, есть обратный элемент
    для любого ненулевого элемента, есть 0 != 1 - нейтральный по умножению; умножение
    дистрибутивно по сложению.

18. Важный пример: поле комплексных чисел

    С = {(a, b) = a + b*i: a, b in R, i^2 = -1}
    +: CxC -> C, (a1, b1) + (a2, b2) |-> (a1 + a2, b1 + b2)
    *: CxC -> C, (a1, b1)*(a2, b2) = (a1 + b1*i)*(a2 + b2*i) = (a1*a2 - b1*b2) +
        + (a1*b2 + a2*b1)*i |-> (a1*a2 - b1*b2, a1*b2 + a2*b1)
    *: RxC -> C, r*(a, b) |-> (r*a, r*b)
    Сложение коммутативно, т.к. сложение в R коммутативно
    Сложение ассоциативно, т.к. сложение в R ассоциативно
    Есть нейтральный элемент по сложению - (0, 0)
    Противоположный по сложению - forall (a, b) in C: (-a, -b)
    Умножение коммутативно и ассоциативно, т.к. умножение в R обладает этими св-ми
    Есть нейтральный элемент по умножению: (1, 0)
    Есть обратный по умножению для всех, кроме (0, 0):
        (a, b)^-1 = (a, -b)/|(a, b)| = (a, -b)/(a^2 + b^2) = (a, -b)/((a, -b)*(a, b))
    Умножение дистрибутивно по сложению:
        (a1, b1)*((a2, b2) + (a3, b3)) = (a1, b1)*(a2 + a3, b2 + b3)
        (a1, b1)*((a2, b2) + (a3, b3)) = (a1*a2 - b1*b2, a1*b2 + a2*b1) +
        (a1*a3 - b1*b3, a1*b3 + a3*b1) = (a1*(a2 + a3) - b1*(b2 + b3), a1*(b2 + b3) + (a2 + a3)*b1)
        = (a1, b1)*(a2 + a3, b2 + b3)

19. Важный пример: кольцо многочленов

    Пусть K - поле. Многочленом от 1 переменной называется бесконечная сумма вида:
        a0 + a1*x^1 + a2*x^2 + ... + an*x^n + ..., где ai in K, x - формальная переменная
    Множество всех многочленов от переменной x будем обозначать K[x].
    Определены операции + и *:
        Пусть f, g in K[x],
            f = +(0, +inf) ai*x^i
            g = +(0, +inf) bi*x^i
        +: K[x]^2 -> K[x]
            Тогда f + g = +(0, +inf)((ai + bi)*x^i) = h
        *: K[x]^2 -> K[x]
            Тогда f*g = +(i = 0, +inf) (+(j = 0, i)(a[j] + b[i - j])*x^i)
    Тогда K[t] - коммутативное кольцо с 1:
        1) f + g = g + f
        2) (f + g) + h = f + (g + h)
        3) exist 0 = +(0, inf) 0*x^i
        4) exist -f = +(0, inf) -a[i]*x^i
        5) f*g = +(i = 0, +inf) (+(j = 0, i)(a[j] + b[i - j])*x^i) =
            +(i = 0, +inf) (+(j = 0, i)(a[j] + b[i - j])*x^i) =
            +(i = 0, +inf) ((+(j = 0, i)a[j] + +(j = 0, i)b[i - j])*x^i) =
            +(i = 0, +inf) ((+(j = 0, i)a[j] + +(j = 0, i)b[j])*x^i) = g*f
        6) (f + g)*h = +(0, inf) d[i]*x^i
            f*h = +(0, inf) p[i]*x^i
            g*h = +(0, inf) q[i]*x^i
            d[i] = +(j = 0, i)((a[j] + b[j])*c[i - j]) = +(j = 0, i)(a[j]*c[i - j]) +
            + +(j = 0, i)(b[j]*c[i - j]) = p[i] + q[i]
        7) ассоц *
        8) наличие 1: a0 = 1, ai = 0, i >= 1

21. Понятие системы линейных алгебраических уравнений

    Линейным уравнением называется уравнение вида a1x1 + .. + anxn = b, где все элементы из поля K.
    Решением уравнения называется набор x1, .., xn, подставив который в уравнение мы получим тождество.
    Системой линейный уравнений называется набор из m уравнений и n неизвестных вида
    {
        a11x11 + ... + a1nx1n = b1
        a21x21 + ... + a2nx2n = b2
        ...
        am1xm1 + ... + amnxmn = bm
    }
    Решением такой системы называется набор, который будет решением каждого отдельного уравнения
    Матрица

22. Внешний закон композиции: определение, примеры

    Внешним законом композиции называется отображение f: AxM -> M,
    а значение (a, x) -> y - композицией a и x относительно закона f.
    При этом элементы из A - операторы закона.

    Примеры:
        1) (Z, '+'), f: NxZ -> Z, f(n, z) = nz = z + ... + z (n раз)
        2) В примере 1 внешний закон f согласован с внутренним законом '+',
        потому что выполняется f(n, z1 + z2) = n(z1 + z2) = nz1 + nz2, а также
        N обладает структурой моноида по умножению, поэтому (n1*n2)(z1 + z2) =
        = n1n2z1 + n1n2z2.
        3) Пусть X - моноид. Определим внешний закон f: NxX -> X следующим
        образом: f(n, x) = x*x*..*x(n раз) = x^n. В частности, x^0 = e, x^1 = x,
        x^2 = x*x, ... Тогда можно задать внутренний закон на множестве X вида
        x^n*x^m = x^(n + m), (x^n)^m = x^(n*m)

23. Понятие согласованно действующей структуры

    Алгебраическая структура A действует на структуре М, если каждый элемент
    А является оператором внешнего закона из М, а также имеет место (a*b)(x) =
    = a(b(x)), forall x in M, a, b in A.

    Алгебраическая структура A согласованно действует на структуру M, если:
        (a*b)(x+y) = a(b(x+y)) = a(b(x)+b(y)) = a(b(x))+a(b(y))

24. Структура модуля над кольцом: определение, примеры

    Левым модулем на кольцом R или левым R-модулем называется абелева группа (G, +)
    с внешним законом композиции RxG -> G: (r, g) -> rg. При этом верно:
        1) (a + b)x = ax + bx forall a, b in R, x in G
        2) (ab)x = a(bx) forall a, b in R, x in G
            То есть алгебраическая структура R действует на структуре G
        3) a(x + y) = ax + ay forall a in R, x, y in G - согласованность
            внешнего и внутреннего законов композиции

    Аналогично задается правый R-модуль с законом GxR -> G.

    Если определены оба модуля, то это двусторонний модуль.

    Примеры:
        Всякий J - идеал кольца R, есть R-модуль
        Любая абелева группа (G, +) - Z-модуль, ведь можно задать
            (z, g) |-> zg
            0g = e, 1g = g, 2g = g + g, ...
            -1g = -g, -2g = -g + -g, ...
        Пусть K - поле. Тогда определим K^n = {(x1, ..., xn): xi in K}, n in N.
